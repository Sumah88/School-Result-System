# School-Result-System
A comprehensive school result management system designed to streamline the process of recording, calculating, and publishing student academic results. This system automates grade calculations, generates report cards, and provides easy access to student performance data for teachers, administrators, students, and parents.A desktop-based school result management application built using Visual Studio Windows Forms to streamline the recording, calculation, and publishing of student academic results. This user-friendly software automates grade calculations, generates professional report cards, and provides easy access to student performance data for teachers and administrative staff.

Key Features:
Student Management: Add, update, and manage student profiles, class assignments, and personal information Marks Entry: Intuitive interface for entering internal assessments, exam scores, and final grades Automated Calculations: Instant GPA/CGPA calculations, percentage conversions, and grade assignments based on predefined criteria Report Card Generation: Create and print professional-looking report cards with subject-wise performance Database Integration: Secure storage of all student records and results using integrated database Search Functionality: Quick access to student records by name, class, roll number, or registration ID User-Friendly Interface: Clean, organized Windows Forms design for easy navigation and data entry

Built With:
Visual Studio (Windows Forms Application)
C# as the primary programming language .NET Framework
Database: SQL Server, MySQL, SQLite

Purpose:
To digitize and simplify school result management through a reliable desktop application, reducing manual paperwork and ensuring accuracy in academic record-keeping while providing an intuitive experience for non-technical users.
